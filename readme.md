## Build Setup

``` bash
# install dependencies
$ composer install
```

- Copy .env.example to .env file
- Create local MySQL DB
- Enter your MySQL DB credentials in the .env file

``` bash
# generate app key
$ php artisan key:generate

# run migrations
$ php artisan migrate

# seed db with test admin user
$ php artisan db:seed
```
